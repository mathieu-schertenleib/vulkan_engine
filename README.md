# vulkan_engine

Vulkan/GLFW experiments

---

## Building from source

### Prerequisites

- [The Vulkan SDK](https://www.lunarg.com/vulkan-sdk)

### Cloning the project

```
git clone --recursive https://github.com/mathieu-schertenleib/vulkan_engine.git
```

### Building the project

```
cd vulkan_engine
cmake -S . -B build
cmake --build build
```
